Evo OmniVoice Documentation

Evo OmniVoice is a high-performance, multilingual TTS system utilizing the advanced capabilities of the OmniVoice architecture.

1. Voice Generation Modes

Auto (Quick TTS) Generates speech using a high-quality automatic voice. Best for simple and fast text synthesis across multiple languages.
Voice Design (Instruct) Create unique voices using descriptive text prompts.

Example: "female, young adult, british accent, low pitch"

Valid English Attributes: male, female, child, teenager, young adult, middle-aged, elderly, whisper, very low pitch, low pitch, moderate pitch, high pitch, very high pitch, american accent, australian accent, british accent, canadian accent, chinese accent, indian accent, japanese accent, korean accent, portuguese accent, russian accent.
Voice Cloning (Zero-Shot) Upload a 3 to 10-second reference audio file to clone any voice instantly, without any model training required.
Expressive Tags Insert emotion and action tags directly into the text for realistic effects and non-verbal cues.

2. Advanced API (FastAPI)

Endpoint: POST /api/generate

{
  "text": "Hello! [laughter] How can I help you today?",
  "language": "en", // en, de, fr, ja, es, pt, no, sv, da, nl, ar
  "mode": "instruct",  // options: "auto", "clone", "instruct", "design"
  "instruct": "male, deep voice, calm",
  "ref_audio": "BASE64_STRING", // Required for clone mode
  "ref_text": "Optional transcript of reference", // Optional for clone mode
  "num_steps": 20, // 16 (Fast) to 32 (HQ)
  "speed": 1.0,    // 0.5 to 2.0
  "duration": 0    // Fixed length in seconds (0 = auto)
}

3. Non-Verbal Symbols (Emotions)

Insert the following tags into your text for realistic non-verbal sounds and effects:

[laughter] [sigh] [whisper] [confirmation-en] [question-en] [surprise-wa] [surprise-yo] [dissatisfaction-hnn]

4. Multilingual Support

Evo OmniVoice fully supports the following languages natively via the API:

  • English (en)
  • German (de)
  • French (fr)
  • Japanese (ja)
  • Spanish (es)
  • Portuguese (pt)
  • Norwegian (no)
  • Swedish (sv)
  • Danish (da)
  • Dutch (nl)
  • Arabic (ar)
